§01 Pair

  • Activate Sys layer Hold both inner thumbs
  • Select Bluetooth slot 0–3 One press selects, then pair
  • Clear current slot Erases pairing on active slot
  • Soft reset Reboot half (Sys layer)
  • Bootloader Flash mode (Sys layer)

On the phone: open Bluetooth, pair the device named SS-<hash>, then mark it as an Input device.

  • Hide the on-screen keyboard Settings

§03 Navigation

  • Home Cmd + Enter
  • Back chord: Esc
  • Recents Cmd + (chord: Tab) Cmd +
  • App switch Alt + (chord: Tab) Alt +
  • Notifications Cmd + N
  • Search / launch app Cmd + F

§04 Window

  • Snap left / right Cmd + ← / →
  • Maximise / minimise Cmd + ↑ / ↓

§05 Screenshot

  • Selection screenshot Cmd + Shift + S

  • Wheel ←/→
  • Wheel ↑/↓
